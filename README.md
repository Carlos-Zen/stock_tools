# A股股票异动监管建议工具

这是一个使用Python开发的A股股票分析工具，能够根据输入的股票代码计算出10天和30天相对大盘偏离值，并给出后续3天的异动监管建议。

## 功能特点

- 计算股票相对于大盘的10天和30天偏离值
- 提供专业的异动监管建议
- 支持GUI界面和命令行两种使用方式
- 使用akshare获取实时股票数据

## 依赖库

- `customtkinter`: 用于创建现代化的GUI界面
- `akshare`: 用于获取A股股票和指数数据
- `pandas`: 用于数据处理
- `numpy`: 用于数值计算

## 安装依赖

```bash
pip install customtkinter akshare pandas numpy
```

## 使用方法

### GUI版本

```bash
python stock_analyzer.py
```

在图形界面中输入股票代码，点击"开始分析"按钮即可。

### 命令行版本

```bash
python stock_analyzer_cli.py
```

按提示输入股票代码即可。

## 功能说明

1. **10天偏离值**: 计算股票最近10个交易日相对于大盘的平均偏离程度
2. **30天偏离值**: 计算股票最近30个交易日相对于大盘的平均偏离程度
3. **异动监管建议**: 根据偏离值大小和方向，提供未来3天的监管风险建议

## 偏离值计算方法

偏离值 = 股票收益率 - 大盘收益率

## 异动监管建议规则

- **偏离度较大 (>5%)**: 提示监管风险，关注资金流向
- **偏离度中等 (2-5%)**: 继续观察，关注后续走势
- **偏离度较小 (<2%)**: 风险较低，走势稳定

## 注意事项

- 本工具仅供学习和参考使用
- 股票投资有风险，请谨慎决策
